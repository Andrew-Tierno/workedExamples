In 1724, physicist Daniel Gabriel Fahrenheit proposed a temperature scale, the Fahrenheit scale.
The Celsius scale is also a temperature scale which serves as an SI derived unit, and hence used worldwide.


Some countries still continue to use the Fahrenheit scale over the Celsius scale.
Write a program that takes in a choice from the user, fahrenheit to celsius or vice versa.
Read in the appropriate temperature and output the converted temperature.


Use the following formulae for the conversions:
celsius = (fahrenheit - 32) * 5/9
fahrenheit = (celsius * 9/5) + 32


NOTE: Take in a real number from the user and not an integer for the temperature.
