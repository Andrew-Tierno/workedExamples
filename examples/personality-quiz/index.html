Since quarantine began, my Facebook has been taken over by various logical and astrological (just teasing!) quizzes. However, none of the quizzes were about a very important topic... me. That's why I came up with my very own personality quiz! You should have all the tools you need to build your own. We'll even include a correct-answer-counter to make the quiz more competitive--then you'll know who really knows you best.

Write a program that asks the user multiple choice questions about you (or anything, really). The quiz should include at least 3 questions that have at least 3 choices each.

  • The user should be instructed how to enter their answer.
  • After each question, the user should be told if they got the question right or wrong.
  • At the end of the quiz, the user should be told their score (the number of correct answers).
  • Each run of the program only needs to play the quiz once.
  • You can assume the user will only enter valid guesses for each question.

Welcome to the Quiz! How well do you know Daniel?
Instructions: select your answer by inputing a, b, or c.
########## 1 ##########
Question 1: what pet does Daniel have?
a. Dog
b. Cat
c. Snake
Type a, b, or c and hit enter: a
That's incorrect...
########## 2 ##########
Question 2: which does Daniel hate the most?
a. Hangnails
b. Waking up early
c. Stories with no emotional resolution
Type a, b, or c and hit enter: c
That's right!
########## 3 ##########
Question 3: what is Daniel doing on a Sunday morning?
a. Sleeping in
b. Eating homemade brunch
c. Working out
Type a, b, or c and hit enter: a

--------------------
That's the end of our quiz!
Congratulations! You got 2 questions right!

For clarity, I suggest keeping your Question/Answer format as multiple choice. However, there's nothing stopping you from including True/False or Fill-In-The-Blank questions! We'll count that as extra credit.

I hope you have fun making this and sharing this with your friends and family!

Once you've learned a bit more, this quiz is a great place for some easy ~extensions~!! Here are some ways you can expand your quiz down the line:

  • The quiz can be run multiple times
  • The name and score of players is recorded on a Leaderboard
  • The order of the answers is shuffled to prevent easy memorization
  • Each quiz consists of 10 questions randomly chosen from a list of 50+
  • Bonus points are earned by correctly answering questions quickly
